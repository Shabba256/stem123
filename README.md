# StreamFlix Firebase + Cloudinary

Netflix-style video platform with:

- Firebase Authentication (admin only)
- Firestore metadata (multi-user)
- Cloudinary video uploads (streaming + download)
- Netflix-style client UI
- GitHub Pages compatible
